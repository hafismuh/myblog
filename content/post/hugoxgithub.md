Hugo adalah static site generator open-source yang memudahkan membangun wesite dari biasanya. Hugo merupakan framework dari bahasa pemrograman Go, atau biasa disebut Golang.

Blog ini dibangun dengan Hugo dan saya tidak perlu memahami lebih dalam Golang karena terdapat banyak tema yang bisa digunakan. Blog ini di-hosting di Github Pages, yang mana memerlukan akun Github. Satu lagi, untuk mendeploy Blog ini saya menggunakan Git version control.

Jadi, hal yang diperlukan adalah:
* Install Git
* Install Hugo
* Membuat akun Github
