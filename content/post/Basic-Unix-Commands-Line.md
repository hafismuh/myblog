CLI merupakan suatu program yang digunakan untuk menjalankan suatu perintah berbasis text.

<H3>Syntax Command</H3>
<ul>
    <li>pwd -> untuk mengetahui posisi direktori saat ini.</li>
    <li>ls -> untuk melihat isi direktori</li>
    <li>cd -> berpindah direktori</li>
        <ul>
            <li>cd ./direktori -> berpindah ke di</li>
            <li>cd .. -> kembali ke direktori sebelumnya</li>
        </ul>
    <li>cat -> melihat isi file</li>
    <li>touch -> membuat sebuah file</li>
    <li>mkdir -> membuat sebuah direktori</li>
    <li>cp -> copy</li>
        <ul>
            <li>cp direktori tujuan</li>
        </ul>
    <li>mv -> move</li>
        <ul>
            <li>mv direktori tujuan</li>
            <li>mv -D ubah nama</li>
        </ul>
    <li>rm -> remove</li>
</ul>
